# Local Plant Species Identification #

The aim of the proposed project is to rely on machine learning approaches to create a reliable and effective system for effortlessly recognizing plant species. Leveraging advancements in machine learning, this project aims to create a scalable and user-friendly solution to empower both researchers and the general public in identifying plant species.
